# Arbitrage Testing Platform

## Project Documentation

This project is designed to facilitate testing of arbitrage strategies in various markets. It includes tools for simulating trades and analyzing the effectiveness of different strategies.

## Setup Instructions

1. **Clone the Repository**
   ```bash
   git clone https://github.com/ifoxwitu329/arbitrage-testing-platform.git
   cd arbitrage-testing-platform
   ```

2. **Install Dependencies**
   Make sure you have Python 3.x installed. Then run:
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the Application**
   To start the application, execute:
   ```bash
   python main.py
   ```

4. **Configuration**
   Update the `config.json` file with your trading parameters.

5. **Testing**
   Use the following command to run tests:
   ```bash
   python -m unittest
   ```
